# Tech Survey Dashboard 2024

## Описание
Интерактивный дашборд для анализа технологических предпочтений 18,845 разработчиков.

## Структура
- Главная Обзор и навигация
- Страница 1 Текущее использование технологий
- Страница 2 Будущие технологические тренды
- Страница 3 Демография респондентов

## Источники данных
- BigQuery проект surveydata-478616
- Dataset tech_survey_data
- 9 таблиц + 13 views

## Как использовать
1. Используйте фильтры для детального анализа
2. Кликайте на элементы графиков для drill-down
3. Наводите курсор для подробной информации
