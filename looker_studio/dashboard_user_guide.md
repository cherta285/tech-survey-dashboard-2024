# Dashboard User Guide

## How to Use the Dashboard

### Filters
Use dropdown filters to:
- Filter by country
- Filter by age group
- Filter by education level

### Navigation
Click tabs at the top:
- Page 1: Current Usage
- Page 2: Future Trends
- Page 3: Demographics

### Interactivity
- Click on chart elements to drill down
- Hover over data points for details
- Export data using the menu
